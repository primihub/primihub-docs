# 匿踪查询
在导航栏点击「匿踪查询」可进入【匿踪查询】页面。

1、先选择中心节点下的机构

![选择中心节点下的机构](/img/pir1.png) 

2、选择机构后，点击查询按钮，弹出机构下的资源，选择资源

![pir2](/img/pir2.png) 
![pir3](/img/pir3.png) 

3、输入检索 ID（ID输入数字且不要超出样本量减去 2 得到的数值，若查询多个可用英文逗号隔开）

![pir4](/img/pir4.png) 

4、点击查询按钮，可进行匿踪查询，可得到结果文件，文件支持下载查看。
