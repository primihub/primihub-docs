开源隐私计算平台 PrimiHub [v1.2.0](https://github.com/primihub/primihub/releases/tag/1.2.0) 发布了！

<!--truncate-->

## Feature

1. Integrate LibPSI 
2. Flight Dataset server
3. TEE (Gramine)
4. Python SDK: 
  * Data client
  * Basic MPC API support: remote function, dataset definition, channel, data reference，get result.
5. FL improvement.
